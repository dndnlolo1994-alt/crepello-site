A small uppercase mono pill for menu tagging and meta labels.

```jsx
<Tag tone="dessert">Sweet</Tag>
<Tag tone="savory">Savory</Tag>
<Tag tone="accent" outline>New</Tag>
```

Tones: `neutral | accent | dessert | savory`. `outline` drops the fill. For numbered section eyebrows (`01 — DRINKS`) use the `<Eyebrow>` component instead.
