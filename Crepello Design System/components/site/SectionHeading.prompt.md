Editorial section header — numbered eyebrow, serif heading, optional lead.

```jsx
<SectionHeading
  num="02"
  eyebrow="Dessert"
  title="Delight in every bite"
  lead="Toasted caramel, warm crepe, slow cream — built to order."
  align="center"
/>
```

Composes `Eyebrow`. Lead is capped at ~46ch. Use `as` to set the heading level for document semantics.
